## Description

Website Calle adalah website sederhana berbasis HTML, CSS, dan JavaScript yang dibuat untuk latihan pengembangan web. Website ini menampilkan produk fashion dengan tampilan modern dan terstruktur.

Pengguna dapat melihat daftar produk, detail produk, serta informasi tambahan seperti harga dan stok.

---

## Features

- Halaman utama (Homepage)
- Navigasi antar halaman
- Halaman daftar produk
- Halaman detail produk
- Komponen header dan footer terpisah
- Tampilan menggunakan Bootstrap
- Layout responsif dasar

---

## Installation

1. Download atau buka folder project
2. Pastikan semua file berada dalam satu folder
3. Buka file `index.html`
4. Website akan langsung berjalan di browser

---

## Usage

1. Buka halaman utama melalui `index.html`
2. Gunakan menu navigasi untuk berpindah halaman
3. Pilih produk untuk melihat detail
4. Gunakan tombol yang tersedia untuk interaksi
